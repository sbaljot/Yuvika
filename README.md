# Project Title: TBD Site

## Description
This project is a simple web application consisting of two HTML pages. The main page (`index.html`) allows users to enter their username, while the second page (`second-page.html`) provides additional content. The project includes CSS for styling and JavaScript for interactivity.

## File Structure
```
tbd-site
├── index.html
├── second-page.html
├── css
│   └── styles.css
├── js
│   └── main.js
├── README.md
└── .gitignore
```

## Getting Started

### Prerequisites
- A web browser to view the HTML pages.
- A code editor (optional) for making changes to the files.

### Installation
1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd tbd-site
   ```

### Running the Project
- Open `index.html` in your web browser to view the main page.
- Use the navigation to go to `second-page.html`.

### Usage
- On the main page, enter your username and click "Submit" to see the corresponding message.
- You can navigate to the second page using the provided links.
